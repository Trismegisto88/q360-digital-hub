Workflows n8n

Patrón estándar: TRIGGER → EXTRACT → TRANSFORM → LOAD → NOTIFY. Motor self-hosted en Digital Ocean + Easypanel. 20 workflows en 4 bloques.

SYNC
NetSuite → Monday (Saldos)
  • NetSuite API → Extraer saldos CxC
  • Transformar → Mapear a columnas Monday
  • Actualizar board Facturación & Cobranza
⏰ Diario 8:00am
SYNC
NetSuite → Monday (Cobranza)
  • NetSuite API → Status de pagos
  • Calcular días mora, aging buckets
  • Actualizar status cobranza en Monday
⏰ Cada 4 horas
SYNC
Sesame → Monday (Asistencia)
  • Sesame API → Check-in/out diario
  • Calcular horas, detectar ausencias
  • Sincronizar con board RH en Monday
⏰ Diario 10:00am
SYNC
Samsara → Monday (GPS/Flota)
  • Samsara API → Ubicaciones, alertas
  • Mapear unidades a CEDIS asignados
  • Actualizar board Flota en Monday
⏰ Cada 30 min
ALERTA
Monitor SLA OTs
  • Monday API → OTs abiertas >4h
  • Clasificar severidad (media/alta/crítica)
  • Enviar alerta WhatsApp + Monday @owner
🔥 Event-driven (continuo)
ALERTA
Geofence Samsara
  • Samsara webhook → Entrada/salida CEDIS
  • Cruzar con calendario asignaciones
  • Alerta si unidad fuera de ruta
🔥 Webhook (tiempo real)
REPORTE
Dashboard CEO Diario
  • Consolidar KPIs de todos los boards
  • Generar snapshot ejecutivo
  • Enviar resumen vía WhatsApp + Email
⏰ Diario 7:00pm
COMPUTE
P&L por Célula
  • NetSuite → Ingresos + Costos por célula
  • Calcular margen, EBITDA, rentabilidad
  • Actualizar Power BI + Monday Finanzas

Principios
Principios de Diseño del SND

Reglas no-negociables que gobiernan toda la arquitectura digital.

🎯
Single Source of Truth
Monday.com es la fuente única. Toda data vive o se refleja ahí.
🔄
Cero Entrada Manual
Si un dato se puede automatizar, se automatiza. Sin excepciones.
🤖
El SND valida el SND
Cero auditorías humanas. Los datos se cruzan y validan solos.
📸
Sin registro, no existe
Evidencia fotográfica, recibos, calendario. Todo documentado.
⚡
Automatizar antes de vender
La prioridad no es vender más, sino automatizar lo que ya tenemos.
Escalamiento
Sistema de Alertas

Escalamiento automático por severidad. Canales: WhatsApp (Evolution API), Email, Monday @menciones.

⚠️ Severidad Media
  • OT sin actualización >2h
  • Técnico sin check-in a las 9:00am
  • Vehículo fuera de geofence >30min
  • Material solicitado sin confirmar >24h
Canal: Monday @owner · SLA: 2h respuesta
🔶 Severidad Alta
  • OT abierta >4h sin resolución
  • Técnico ausente sin justificación
  • Reclamo de cliente recibido
  • Factura vencida >30 días sin gestión
Canal: WhatsApp + Monday · SLA: 1h respuesta
🔴 Severidad Crítica
  • Incidente de seguridad en CEDIS
  • Cliente escala a nivel corporativo
  • Sistema core caído (Monday/n8n/NetSuite)
  • Accidente vehicular o laboral
Canal: WhatsApp + Llamada + Email CEO · SLA: 15min
